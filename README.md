# fast-loan-app
## App for taking fast loans
Main features:
1. User Service
 - Register
 - Find
 - Delete
2. Application Service
- Apply
- Verify
- Find
- Delete
3. Loan Service
- Create
- Extend
- Calculate cost
- Find
- Delete
###### Built with:
* Spring v2.2.1.RELEASE
* Hibernate
* Gradle v. 6.0
* MySQL v. 8.0.19
